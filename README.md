# 🌍 Project: Deity (GudSim)
**The Ultimate God Simulator for Mobile**

Цей проект — духовний спадкоємець Godus, але без нудного клікання та з реальною свободою. 

### 🔥 Чому ми кращі за оригінал?
* **Smart Settlers:** Поселенці мають власний розклад дня та потреби.
* **True Terraforming:** Деформація ландшафту оптимізована під сенсорні екрани.
* **Elemental Magic:** Блискавки, метеорити та цунамі, що реально впливають на фізику світу.
* **Open Source:** Розвиваємо гру разом зі спільнотою!

### 🛠 Стек технологій
* **Engine:** Godot 4.3 (Mobile Renderer)
* **Terrain:** TerraBrush Core
* **Language:** GDScript
